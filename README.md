# 📱 Lattes de Bolso

> Aplicativo mobile de currículo digital pessoal, desenvolvido com React Native e Expo.

---

## 🗂️ Sobre

O **Lattes de Bolso** é um app que apresenta informações acadêmicas e técnicas de forma interativa no celular, inspirado na plataforma Lattes do CNPq.

---

## 🛠️ Tecnologias

- ⚛️ [React Native](https://reactnative.dev/) — Framework principal
- 🚀 [Expo](https://expo.dev/) — Plataforma de desenvolvimento
- 🧭 [React Navigation](https://reactnavigation.org/) — Navegação entre telas
- 🎨 Expo Vector Icons — Ícones (Ionicons + MaterialCommunityIcons)

---

## 🖥️ Telas

| 🗺️ Tela | 📝 Descrição |
|--------|-------------|
| 🏠 **Home** | Menu principal de navegação |
| 👤 **Perfil** | Dados pessoais e tecnologias |
| 🎓 **Formação Acadêmica** | Graduação e pós-graduação |
| 📜 **Cursos** | Certificados e capacitações |
| 💡 **Habilidades** | Competências técnicas por categoria |
| 📚 **Conhecimentos** | Áreas de domínio |
| 🔗 **Links** | Lattes, LinkedIn e portfólio |

## ⚙️ Como Executar

### ✅ Pré-requisitos

- 🟢 [Node.js](https://nodejs.org/) instalado
- 📦 [Expo CLI](https://docs.expo.dev/get-started/installation/) instalado
- 📲 Aplicativo **Expo Go** no celular

### ▶️ Passo a passo

```bash
# 📥 Instale as dependências
npm install

# 🚀 Inicie o app
npx expo start
```

Escaneie o QR Code com o **Expo Go** no celular, ou pressione:

- 🤖 `a` — Android
- 🍎 `i` — iOS
- 🌐 `w` — Web
